# My Trail Maps

A collection of high-quality off-road trail maps for GPS navigation. All maps are compatible with popular GPS apps like Gaia GPS, Polaris Ride Command, and other navigation applications.

## Repository Structure

```
/
├── index.html           # Homepage with site overview
├── maps.html            # Complete maps gallery with all trail maps
├── trail-maps/          # Directory containing all trail map images
└── images/              # Additional images and assets
```

## Features

- **37+ Trail Maps** covering popular off-road parks across the United States
- **GPS Compatible** - All maps work with Gaia GPS, Polaris Ride Command, and other GPS apps
- **Clean, Modern Interface** - Easy-to-navigate gallery with detailed map information
- **Mobile Responsive** - Works perfectly on all devices

## Usage

### Viewing the Maps

1. Open `index.html` in a web browser to see the homepage
2. Click "Browse Trail Maps" to view all available maps
3. Click on any map card to view the full-size trail map
4. Maps can be downloaded for use with GPS devices

### GPS Integration

The trail maps in this repository are designed to work with:
- Gaia GPS App
- Polaris Ride Command
- BackCountry Navigator
- onX Offroad
- Other GPS navigation apps supporting standard map formats

## File Organization

- **trail-maps/** - Contains 97 PNG format trail map files
- **images/** - Contains 143 supporting images and graphics
